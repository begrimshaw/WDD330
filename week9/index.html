Chapter 14: HTML5 API

  • Newest version of HTMl
  • data- attribute will allow you to embed data in a web page using custom attributes that are ignored by the browser. Used to identify particular elements. Ex: data-dropdown could identify the dropdown menu
  • Ability to use local and session storage
  • Geolocation API can obtain the geographical position of the device. Can be used to find the user's exact location then link to nearby placfes or measure the speed that the user is moving. Permission to use this must first be granted by the user.
  • Geolocation is a property of the navigator object. Ex: navigator.geolocation.getCurrentPosition(youAreHere); With the youAreHere being a callback function after retrieving the position
  • The position can then return the longitutde and latitude, the speed, the altitude, heading, and timestap of the device.
