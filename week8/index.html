Chapter 8: Transofrms and Transitions

  • Off the top of my head, I have used transform and transitions a couple of times. I've used it to transform hamburger menu's into an 'x' to close.
  • Transform property- translate, rotate, scale, and skew any element on the page
  • Translation function- move elements left, right, up or down. Similar to position: relative
  • Scaling function- scales an element by the defined factors horizontally then vertically. "transform: scale(1.5, 0.25);"
  • Rotate function- rotates an element around the point of orgin by a specified degree. "transform: rotate(10deg)"
  • Transition- Allow values of CSS properties to change over time. (used for buttons a lot to fade in a color). Has a duration often times.
  • We can use multiple transition properties on one element. Ex: One to change the color and one to rotate
  • Animations- done using keyframes. Is faster in CSS than using javascript behing the scenes
  • Keyframes- Name the animation then attach it to an element. Ex: "@keyframes myAnimation { /* put animation keyframes here */ }"
  • Examples: @keyframes moveRight { from { transform: translateX(-50%); } to { transform: translateX(50%); } }
    @keyframes appearDisappear { 0%, 100% { opacity: 0; } 20%, 80% { opacity: 1; } }
